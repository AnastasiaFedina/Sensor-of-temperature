# Sensor-of-temperature
Система, яка складається з головного(Master) та підлеглих(Slave) мікроконтроллерів, 
що отримують та обробляють данні про температуру навколишнього середовища. 
Мікроконтроллери Slave отримують інформацію з датчиків температури та передають 
її через UART на Master. Master обробляє ці данні та передає їх на коп'ютер користувача 
через COM-порт.
